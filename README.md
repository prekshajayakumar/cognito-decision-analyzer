# Cognito — Decision Style Analyzer

Cognito is a Django-based cognitive science project that analyzes user decision-making behavior through a quiz interface. It records response time and hesitation patterns, then classifies users as impulsive, analytical, or balanced decision-makers.

## Cognitive Science Basis
The project is based on dual-process theory:
- Fast thinking (intuitive, impulsive)
- Slow thinking (deliberate, analytical)

## Features
- Timed quiz interface
- Response tracking
- Hesitation detection
- Rule-based decision style classification
- Django admin for stored sessions and responses

## Tech Stack
- Django
- Python
- SQLite
